# LSI Models Overview

This folder contains the mathematical and algorithmic models behind Liquidity Sequencing Infrastructure (LSI).

## Contents
- **sequencing-graph/** — Graph-based liquidity sequencing models.  
- **timing-models/** — Liquidity-time curve, programmable windows, timing functions.  
- **flow-optimization/** — Flow Optimization Engine (FOE) logic and algorithms.  
- **simulation-outputs/** — Outputs from model runs (CSV, images, metrics).  

Each model includes:
- purpose  
- assumptions  
- mathematical formulation  
- pseudo-code  
- references to diagrams
