# Changelog
All notable changes to this project will be documented in this file.

The format is simple and human-first.

## [0.1.0] - Initial Research Scaffold
- Created core repository structure:
  - `/papers` for abstracts, drafts, and final papers
  - `/src/models`, `/src/architecture`, `/src/simulations`
  - `/notes` for ideas, research notes, and scratch work
  - `/diagrams` for all visual artifacts
- Added project governance and meta files:
  - `README.md`
  - `LICENSE`
  - `CONTRIBUTING.md`
  - `AUTHORS.md`
  - `CODE_OF_CONDUCT.md`
  - `SECURITY.md`
  - `.gitignore`
- Added templates for:
  - abstract, draft, and final papers
  - models and architecture docs
  - simulation scenarios and runner notes
  - notes and diagrams placeholders
